# What is Git?

### Git is a _*distributed*_ _*version control system*_

## version control system
* Keeps track of changes
* Fixing mistakes is easy

## distributed
* Works with teams
* Everyone is a peer: no bosses

## What is Github?
* Just a site to make sharing easy
